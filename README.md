## 📋 Deskripsi Proyek

Proyek ini bertujuan untuk memprediksi nilai ujian siswa berdasarkan durasi belajar menggunakan algoritma **Regresi Linear** dan **Polynomial Regression**. Model ini dapat membantu siswa, guru, dan orang tua dalam memahami hubungan antara jam belajar dan prestasi akademik.

### Domain Proyek
- **Bidang:** Pendidikan / Educational Data Mining
- **Masalah:** Banyak siswa belum mengetahui efektivitas durasi belajar terhadap hasil ujian
- **Solusi:** Mengembangkan model machine learning untuk memprediksi nilai ujian berdasarkan jam belajar
